# TLC-player
Minimalist, Fast, FOSS, Terminal-Based Music Player That can give you Your loved music's lyrics, Right at your terminal !

## Usage :
* Installing dependencies :
```bash
pip3 install -r requirements.txt
```
* Run TLC with :
```bash 
python3 tlc.py <music-path>
```
